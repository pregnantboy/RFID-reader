processReceivedLine:moreLinesAvailable:

Each correctly terminated line from the device is passed to this method for processing

- (BOOL)processReceivedLine:(NSString *)fullLine moreLinesAvailable:(BOOL)moreAvailable

Parameters

fullLine

The line to be processed

moreAvailable

When YES indictates there are additional lines to be processed (and will also be passed to this method)

Return Value

YES if this line should not be passed to any other responder
