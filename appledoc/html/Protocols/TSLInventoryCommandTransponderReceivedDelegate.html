transponderReceived:crc:pc:rssi:fastId:moreAvailable:

The transponder information received

- (void)transponderReceived:(NSString *)epc crc:(NSNumber *)crc pc:(NSNumber *)pc rssi:(NSNumber *)rssi fastId:(NSData *)fastId moreAvailable:(BOOL)moreAvailable

Parameters

epc
  • the transponder identifier as hex-encoded, ascii string
crc
  • the transponder CRC as unsigned short encoded in NSNumber or nil if none
pc
  • the transponder program control word as unsigned short encoded in NSNumber or nil if none
rssi
  • the transponders RSSI value (in dBm) as int encoded in NSNumber or nil if none
fastId
  • the transponders TID bank as NSData or nil if none (Impinj Only - requires transponder support)
moreAvailable
  • YES if there are more transponders to be delivered
