Properties

lastCommandLine

The last command line sent

@property (nonatomic, readonly) NSString *lastCommandLine

Discussion

Because TSLAsciiCommands add a unique id number each time their commandLine property is accessed
this property can be used (primarily for debugging) to get the actual command line issued (if examined
immediately after the command is executed)

executeCommand:withResponse:

Executes the given (synchronous) command using the supplied response lines

- (void)executeCommand:(id<TSLAsciiCommand>)command withResponse:(NSArray *)responseLines

Parameters

command

the command to ‘execute’

responseLines

the response to feed through the responder chain

Discussion

Responses must OMIT the first CS: line but must include one of ER: or OK: as the last line.
The CS: line will be supplied from the command’s command line
