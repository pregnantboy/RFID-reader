Detailed Description

The interface for classes that handle responses from ASCII 2.0 commands

Method Documentation

- (void) clearLastResponse

Clears the values from the last response

Derived classes must call super class to ensure correct operation

- (BOOL) processReceivedLine: (NSString *)  fullLine
moreLinesAvailable: (BOOL)  moreAvailable 

Each correctly terminated line from the device is passed to this method for processing

Parameters
fullLineThe line to be processed
moreAvailableWhen YES indictates there are additional lines to be processed (and will also be passed to this method)
Returns
YES if this line should not be passed to any other responder
